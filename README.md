# 🛡️ Heimdall Ops

O Heimdall Ops é uma ferramenta desenvolvida para digitalizar e padronizar as rondas operacionais e de infraestrutura. Ele elimina o uso de papel, permitindo o registro de anomalias com fotos e gerando um relatório formal instantaneamente.

### 🎯 Funcionalidades
* Formulário dinâmico para registro de ocorrências durante a ronda.
* Suporte a anexo de imagens para documentação de problemas.
* Geração automática de relatórios em PDF formatados para envio à gerência.
* Histórico de rondas salvo na nuvem.

### 🛠️ Tecnologias
* JavaScript (ES6+)
* Firebase (Armazenamento de dados e imagens)
* PDFMake (Geração de relatórios PDF no client-side)
